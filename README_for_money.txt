***README***

Files included: money.cpp
--------------------------

1. The program contain one file money.cpp
2. money.cpp contains the main function and the two classes i.e. currency_converter and currency_exchange
3. money.cpp contains the definations for all the functions being used.

Compilation instructions:
-------------------------

running money.cpp
g++ money.cpp( make sure 'm' is not capitalized)
./a.out

When run:
---------


Program will ask name of exchange office and the manager. After that, it will ask for your position , one of the east, west, north, and south. Depending on your position, it will print the name of exchange office and it's manager name and ask for amount of input money and currency name. It will print the output amount of dollars equivalent to the input amount of currency if input currency is included for exchange to dollars. The available currency to convert to dollars are pound, peso, yen, rupees, and dinar. If you typed other currency to exchange to dollars other than the above mentioned, it will print message as "We do not convert the given currency here. Sorry". For exit, input "exit", it will print total dollars dispensed by both exchanges and exits the program.
